# Using Large Language Models (LLMs)

LLMs 可用于抽取文献主题、生成摘要、理解用户兴趣，提升推荐质量。

## 推荐模型
- SPECTER / SciBERT（科学文献嵌入）
- OpenAI GPT-4（理解用户输入/偏好）
- LLaMA2 / Mistral（轻量本地推理）

## 应用场景
- 文献摘要生成
- 用户兴趣摘要压缩
- 多文献主题聚类与对比
